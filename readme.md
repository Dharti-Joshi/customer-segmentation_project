# 📌 Project Overview

Customer Segmentation is a Machine Learning project that groups customers based on their purchasing behavior and demographic information.

Using **K-Means Clustering**, this project identifies different customer segments to help businesses understand customer behavior and create targeted marketing strategies.

---

# 🎯 Project Objectives

* Analyze customer demographics and purchasing behavior.
* Segment customers using K-Means Clustering.
* Visualize customer groups.
* Build an interactive Streamlit dashboard.
* Generate business insights for decision-making.
